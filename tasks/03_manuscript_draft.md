## 引用文献管理

### 形式設定
| 項目 | 設定 |
|------|------|
| 引用形式 | Vancouver（数字順） |
| 管理ツール | Zotero / Mendeley / EndNote |
| DOI | 必須 |
| アクセス日 | Web資料の場合必須 |

### 引用文献チェックリスト
- [ ] 参考文献管理ソフトを使用
- [ ] 全ての引用にDOIを付与
- [ ] 本文中の引用番号と文献リストが一致
- [ ] 文献情報の正確性を確認
- [ ] 最新の文献を含む（直近2-3年）
- [ ] 重要な先行研究を網羅
- [ ] 自己引用の割合が適切（通常10-15%以下）

### 引用文献サマリー
```
総引用数: _____ 件
直近5年の文献: _____ 件 (____%)
自己引用: _____ 件 (____%)
```

### Vancouver形式の例
```
1. Author AA, Author BB, Author CC. Title of article. Journal Name. 2024;12(3):123-456. doi:10.1000/xxxxx

2. Author DD, Author EE. Title of book chapter. In: Editor FF, Editor GG, editors. Book Title. City: Publisher; 2024. p. 100-150.
```
